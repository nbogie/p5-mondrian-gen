# p5-mondrian-gen
Very rough Mondrian-inspired image generator.  It doesn't try too hard to imitate his stuff.

[Try it out here](https://nbogie.github.io/p5js-mondrian-gen/)

Uses [p5.js](https://p5js.org/)
